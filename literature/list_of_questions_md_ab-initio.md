# ЛЕКЦІЇ

Лекція 1. Вступ до комп’ютерного матеріалознавства.
1. Історія, задачі, роль моделювання у фізиці твердого тіла.
2. Класи методів: ab іnіtіo, молекулярна динаміка, Монте-Карло, континуальні підходи.
3. Зв'язок потенціальної енергії та сили.
4. Алгоритм Верле.

Лекція 2. Фундаментальні рівняння атомного рівня.
1. Хвильові властивості мікрочастинок. Гіпотеза де Бройля.
2. Хвильова функція, її фізичний зміст та основні властивості.
3. Оператори фізичних величин у квантовій механіці.
4. Повне (часове) рівняння Шредінгера для однієї частинки та системи частинок. 
5. Рівняння Шредінгера для кристала. Апроксимація Борна–Оппенгеймера.

Лекція 3. Методи квантово-механічних розрахунків.
1. Власні функції та власні значення операторів. Виродження.
2. Самоспряжені (ермітові) оператори.
3. Принцип суперпозиції станів у квантовій механіці.
4. Визначення середніх значень фізичних величин у квантовій механіці.
5. Повна похідна оператора за часом. Інтеграли руху квантової механіки.
6. Принцип тотожності частинок. Симетричні та антисиметричні хвильові функції. Закон збереження симетрії.
7. Детермінант Слеттера. Заборона Паулі.
8. Варіаційний метод Рітца.

Лекція 4. Основи теорії функціонала густини (DFT), псевдопотенціали, базиси, енергетичні функціонали. Приклади програмних пакетів (VASP, Quantum ESPRESSO, ABINIT).
1. Визначення електронної густини через хвильову функцію.
2. Перша теорема Хоенберга-Кона.
3. Друга теорема Хоенберга-Кона.
4. Рівняння Кона-Шема.

Лекція 5. Емпіричні потенціали для металів. Підходи EAM, MEAM, Fіnnіs–Sіnclaіr, Gupta. Параметризація, валідація, межі застосування.
Лекція 6. Побудова та масштабування потенціалів. Практика створення міжатомних потенціалів для металевих сплавів.
1. Типи потенціалів міжатомної взаємодії та основні етапи їх створення.
2. Побудова  EAM-потенціалів чистих металів: математична модель, параметризація, валідація.

Тема 7. Моделювання фізичних властивостей металів. Розрахунок густини, коефіцієнтів теплового розширення, пружних констант, дифузії, кореляцій.
1. Створення сценарію LAMMPS для визначення густини матеріалів.
2. Створення сценарію LAMMPS для визначення коефіцієнта теплового розширення матеріалів.
3. Створення сценарію LAMMPS для обчислення пружних сталих матеріалів.
4. Створення сценарію LAMMPS для моделювання дифузії атомів у матеріалах.
5. Термодинамічні потенціали. Перша та друга група співвідношень Максвелла.


Тема 8. Дефекти кристалічної структури та їх динаміка. Вакансії, дислокації, міжвузлові атоми, зернові межі. Дифузія і сегрегація в сплавах.
1. Рівноважна концентрації вакансій у кристалах.
2. Рівноважна концентрація пар Френкеля у кристалах.
3. Створення вакансій та міжвузлових атомів за допомогою сценарію LAMMPS.
4. Створення меж зерен у кристалах за допомогою сценарію LAMMPS.
5. Формула Ейнштейна-Смолуховського для коефіцієнта дифузії.


# ПРАКТИЧНІ ЗАНЯТТЯ

Практичне заняття 1. Геометрія кристалічної решітки. Елементи кристалографії.
1. Трансляційна решітка, решітка з базисом. Вектор трансляції, найменші осьові вектори трансляції.
2. Елементарна комірка. Типи елементарних комірок. Кристалографічні синґонії та решітки Браве.
3. Індекси вузлів, напрямків та площин у кристалах (індекси Міллера).
4. Обернена решітка та її застосування. Властивості оберненої решітки.

Практичне заняття 2. Установка пакетів LAMMPS та Quantum Espresso, структура та запуск сценаріїв.

Практичне заняття 3. Спектральна густина нормальних коливань. Фонони. Теорії теплоємності кристалічної решітки.
1. Нормальні коливання у кристалах, фонони. Температура та частота Дебая.
2. Спектральна густина нормальних коливань.
3. Класична теорія теплоємності твердих тіл. Закон Дюлонга-Пті.
4. Квантова теорія теплоємності твердих тіл Ейнштейна. Температура та частота Ейнштейна.
5. Квантова теорія теплоємності твердих тіл Дебая. 

Практичне заняття 4. Класична та квантові статистики для систем багатьох частинок.
1. Класична cтатистика Максвела-Больцмана.
2. Квантова статистика Фермі-Дірака.
3. Квантова статистика Бозе-Ейнштейна.
4. Вироджені та невироджені системи. Температура виродження.
5. Особливості та описання електронного газу в металах та напівпровідниках.
6. Ефективна маса.
7. Кінетичне рівняння Больцмана.

Практичне заняття 5. Автоматизація молекулярно-динамічного та ab іnіtіo моделювання, обробка одержаних результатів.
1. Для чого потрібна автоматизація молекулярно-динамічного та ab іnіtіo моделювання. 
2. Привести приклад автоматизації молекулярно-динамічного моделювання. 
3. Привести приклад автоматизації обробки результатів молекулярно-динамічного моделювання.


# ЛАБОРАТОРНІ РОБОТИ

Лабораторна робота 1. Визначення сталих кристалічної ґратки та енергії утворення дефектів у металах.
1. Продемонструвати роботу скрипта LAMMPS для визначення сталої кристалічної гратки кристала згідно зі своїм варіантом завдання, порівняти з експериментом та зробити висновки.
2. Продемонструвати роботу скрипта LAMMPS для визначення енергії вакансії у кристалі згідно зі своїм варіантом завдання, порівняти з експериментом та зробити висновки.
3. Продемонструвати роботу скрипта LAMMPS для визначення енергії міжвузлового атома у кристалі згідно зі своїм варіантом завдання, порівняти з експериментом та зробити висновки.

Лабораторна робота 2. Моделювання процесу деформації металевого сплаву та визначення межі міцності матеріалу.
1. Продемонструвати роботу скрипта LAMMPS для моделювання деформації розтягування матеріалу згідно зі своїм варіантом завдання.
2. Продемонструвати роботу скрипта LAMMPS для моделювання деформації стиснення матеріалу згідно зі своїм варіантом завдання.
3. Визначення межі міцності матеріалу згідно зі своїм варіантом завдання, порівняти з експериментом та зробити висновки.
